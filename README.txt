Simple código del Clásico juego de Piedra Papel o Tijera
El jugador debe ingresar los números 1, 2 o 3 de acuerdo a lo que quiera seleccionar:
1 Piedra
2 Papel
3 Tijera

La aplicación lanzará un resultado Aleatorio, luego comparará con el ingreso del Jugador y devolverá un resultado:
GANA USUARIO
PIERDE USUARIO
EMPATE
